Reverse Polish Notation Calculator
Joe Fair

This Reverse Polish Notation (rpn) calculator is written in Java.  The code is plaform-independent, but it is written on Ubuntu 10.4 with Java 1.6.0_20-b02.

USAGE:
Compile:  The compile command is in the "c" script in this directory.
Test: The test command is in the "t" script in this directory.
Run: java RPNCalculator (or the "r" script in this directory)

Inside the calculator, you can print the current stack, clear all the values, or undo.  Use 'h' for help.
